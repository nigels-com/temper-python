# systemd service using fluent-bit

## Configuration

- NAME service name
  default: temper-python
- PREFIX prefix for temper-python and fluent-bit installation
  default: `/usr/local`

## Install

- 'sudo make install'

## Start/stop service

- 'sudo make start'
- 'sudo make stop'

## Service status

- 'make status'
- 'make logs'
